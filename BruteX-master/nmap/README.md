# BruteX v1.1 by 1N3

Automatically brute force all services running on a target including:

- Open ports
- DNS domains
- Web files
- Web directories
- Usernames
- Passwords

USAGE:
./brutex target

DEPENDENCIES:

- NMap
- Hydra
- Wfuzz
- SNMPWalk
- DNSDict

To brute force multiple hosts, use brutex-massscan and include the IP's/hostnames to scan in the targets.txt file.
