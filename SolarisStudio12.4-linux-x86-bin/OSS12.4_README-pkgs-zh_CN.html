======================================
概述
======================================

Oracle Solaris Studio 12.4 是一套编译器和工具，可供开发者创建适用于 Oracle Solaris 和 Linux 操作系统的高度优化且安全的应用程序。这些工具可帮助应用程序开发者在 Oracle 最新的 T 系列和 M 系列 SPARC 服务器、Fujitsu 的 M10 服务器及基于 Intel 的服务器上实现最佳性能。

有关 Oracle Solaris Studio 12.4 发行版的完整信息，请访问：

Oracle Solaris Studio 12.4 文档库

http://docs.oracle.com/cd/E37069_01/

要大致了解 Oracle Solaris Studio 12.4 产品每个组件的概况，请参阅文档库中的以下手册。

Oracle Solaris Studio 12.4：概述

http://docs.oracle.com/cd/E37069_01/html/E37073/index.html

本文档简要介绍了 Oracle Solaris Studio 12.4 产品中的编译器和工具及其入门知识。

Oracle Solaris Studio 12.4 发行版的新增功能

http://docs.oracle.com/cd/E37069_01/html/E37071/index.html

介绍本发行版中的新功能。

======================================
安装说明
======================================

在安装之前，建议阅读《Oracle Solaris Studio 12.4 安装指南》，其中包含在 Oracle Solaris 10、Oracle Solaris 11、Oracle Linux 系统和 Red Hat Linux 系统上安装此产品的全面信息。

http://docs.oracle.com/cd/E37069_01/html/E37072/index.html

======================================
适用于 Solaris 10 的系统软件包和修补程序
======================================

《Oracle Solaris Studio 12.4 发行说明》中的“系统要求”说明了支持的操作系统版本。

有关运行 Oracle Solaris Studio 12.4 所必需的 Oracle Solaris 系统软件包的完整列表，请参阅

《Oracle Solaris Studio 12.4 发行说明》中的“必需的系统软件包”

http://docs.oracle.com/cd/E37069_01/html/E37070/index.html

Oracle Solaris Studio 12.4 产品分发随附了必需的修补程序。

如果未在运行 GUI 安装程序过程中安装随附的 Oracle Solaris 10 修补程序，请以 root 用户身份安装它们：

# ./install_patches.sh

======================================
安装后 - PATH/MANPATH 设置
======================================

由于 Oracle Solaris Studio 12.4 软件产品组件和手册页未安装到系统目录 /usr/bin/ and /usr/share/man 中，因此可能需要更改您的 PATH 和 MANPATH 环境变量以使用 Oracle Solaris Studio 12.4 软件。

请参见以下节：

《Oracle Solaris Studio 12.4 安装指南》中的“设置开发者工具和手册页访问”：

http://docs.oracle.com/cd/E37069_01/html/E37072/index.html
