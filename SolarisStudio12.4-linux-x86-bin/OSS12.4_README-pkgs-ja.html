======================================
概要
======================================

Oracle Solaris Studio 12.4 は、開発者が Oracle Solaris および Linux オペレーティングシステムのために高度に最適化された、セキュアなアプリケーションを作成できるようにするコンパイラとツールのスイートです。これらのツールは、アプリケーション開発者が Oracle の最新の T シリーズおよび M シリーズ SPARC サーバー、富士通の M10 サーバー、Intel ベースのサーバー上で最高のパフォーマンスを実現するのに役立ちます。

Oracle Solaris Studio 12.4 リリースの詳細については、次のサイトを参照してください。

Oracle Solaris Studio 12.4 ドキュメントライブラリ

http://docs.oracle.com/cd/E37069_01/

Oracle Solaris Studio 12.4 製品の各コンポーネントの大まかな概要を確認するには、ドキュメントライブラリにある次のマニュアルを参照してください。

『Oracle Solaris Studio 12.4: 概要』

http://docs.oracle.com/cd/E37069_01/html/E37073/index.html

このドキュメントでは、Oracle Solaris Studio 12.4 製品内のコンパイラとツールの簡単な説明のあと、各ツールの使用を開始する方法について説明しています。

『Oracle Solaris Studio 12.4 リリースの新機能』

http://docs.oracle.com/cd/E37069_01/html/E37071/index.html

このリリースの新機能について説明しています。

======================================
インストール手順
======================================

インストールの前に、Oracle Solaris 10、Oracle Solaris 11、Oracle Linux システム、および Red Hat Linux システムにインストールするための完全なインストール情報が含まれている『Oracle Solaris Studio 12.4: インストールガイド』を参照することをお勧めします。

http://docs.oracle.com/cd/E37069_01/html/E37072/index.html

======================================
Solaris 10 のためのシステムパッケージとパッチ
======================================

『Oracle Solaris Studio 12.4: リリースノート』の「システム要件」には、サポートされているオペレーティングシステムのバージョンが記載されています。

Oracle Solaris Studio 12.4 を実行するために必要な Oracle Solaris システムパッケージの完全なリストについては、次のセクションを確認してください。

『Oracle Solaris Studio 12.4: リリースノート』の「必要なシステムソフトウェアパッケージ」

http://docs.oracle.com/cd/E37069_01/html/E37070/index.html

Oracle Solaris Studio 12.4 製品配布には必須パッチが含まれています。

GUI インストーラの実行中に、含まれている Oracle Solaris 10 パッチをインストールしなかった場合は、root としてそれらのパッチをインストールしてください。

# ./install_patches.sh

======================================
インストール後 - PATH/MANPATH の設定
======================================

Oracle Solaris Studio 12.4 ソフトウェア製品のコンポーネントやマニュアルページはシステムディレクトリ /usr/bin/ および /usr/share/man にインストールされないため、Oracle Solaris Studio 12.4 ソフトウェアを使用するには PATH と MANPATH の環境変数の変更が必要になることがあります。

次のセクションを参照してください。

『Oracle Solaris Studio 12.4: インストールガイド』の「開発ツールとマニュアルページ用の環境変数の設定」

http://docs.oracle.com/cd/E37069_01/html/E37072/index.html
